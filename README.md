# FNAF-DEMO-DS
This is  for a FNaF demake to the DS

The Game ony works on a DSi, DSi XL, or DS lite.

To play the game, you'll need an R4, you'll need to connect it to your computer, then, to put the game in the "GAME" folder

It's that simple

Also this is only a BETA version, so there's more to come, and more games. Contact me if you find any glitch


Have Fun Playing :)
